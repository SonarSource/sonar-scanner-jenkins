This will inject in your build several variables related to the chosen SonarQube installation defined in the system configuration.
The following variables might be set, depending on the configuration:

  • SONAR_HOST_url
  • SONAR_LOGIN
  • SONAR_PASSWORD
  • SONAR_JDBC_URL
  • SONAR_JDBC_USERNAME
  • SONAR_JDBC_PASSWORD
  • SONAR_EXTRA_PROPS
  • SONAR_MAVEN_GOAL

You can use these variables when configuring the build and launch a sonar analysis using various tools such as maven, gradle, ant, sonar-runner or command line executables.
